# Transcriber

This program transcribes .mp4 or .mp3 files to .srt subtitle files, which can then be imported to video editing softwares directly. 

Setup:
```
pip install -r requirements.txt
```
Requirements:
- openai-whisper
- moviepy

Usage:
```
python transcriber.py
```
